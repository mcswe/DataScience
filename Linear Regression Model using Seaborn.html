Today I will build a machine learning model that uses Seaborn's mpg dataset to effectively and accurately predict the miles per gallon column, using data from other columns.

First, I will import the mpg dataset from Seaborn, drop non-numerical columns: origin and name, and then drop any NaN's that may exist.

In [42]:
mpg = sns.load_dataset("mpg")
mpg = mpg.drop(columns = ['origin', 'name'])
mpg = mpg.dropna()

Now I will split my mpg data into train, validation, and test sets of 250, 75, and 67 respectively. I will drop the mpg column (our target variable) for my X sets, and keep only the mpg column for my y sets.

In [43]:
ds = train_test_split(mpg, test_size = 67, random_state=1)

# splits entire df into 67 test and 325 remaining. DO NOT TOUCH!!!
y_test = ds[1]['mpg']
X_test = ds[1].drop(columns='mpg')

ds2 = train_test_split(ds[0], test_size = 250, random_state=1)
# splits the 325 remaining ('train') into 250 train and 142 val
y_val = ds2[0]['mpg']
X_val = ds2[0].drop(columns='mpg')

y_train = ds2[1]['mpg']
X_train = ds2[1].drop(columns='mpg')

Now I want to train a successful linear regression model to predict the mpg column. I will make use of several functions to do so:

  • sklearn linear_model.LinearRegression() initializes a Linear Regression object.
  • predict() predict mpg values based off training and validation data
  • score computes the coefficient of determination
  • mean_squared_error computes the mean squared error (i.e. mean squared deviation)
In [44]:
model = linear_model.LinearRegression()
In [45]:
model_fit = model.fit(X_train,y_train)

y_train_pred = model.predict(X_train)
y_val_pred = model.predict(X_val)

codet_train = model.score(X_train, y_train)
mse_train = mean_squared_error(y_train, y_train_pred, squared=False)

codet_val = model.score(X_val, y_val)
mse_val = mean_squared_error(y_val, y_val_pred, squared=False)

We can see that the mean squared error of our training and validation set aren't the best. We want this value as close to 0 as possible. Additionally, we want the coefficient of determination as close to 1 as possible.

To help the model perform even better, I will perform feature expansion on the mpg dataset. I will make use of PolynomialFeatures which is part of the sklearn.preprocessing.PolynomialFeatures class. This generates a dataframe consisting of all polynomial combinations of features with a degree less than or equal to a specified degree (in this case, a degree of 2).

In doing so, I will fit this data to a new linear regression model, as well as compute new coefficients of determination and mean squared errors of the new data.

In [47]:
poly = PolynomialFeatures(2, interaction_only=True)
In [48]:
poly_xtrain = pd.DataFrame(poly.fit_transform(X_train))
poly_xval = pd.DataFrame(poly.fit_transform(X_val))
In [49]:
polymodel = linear_model.LinearRegression()
model2_fit = polymodel.fit(poly_xtrain, y_train)

Validation¶

Now I will perform some "sanity checks" to help validate my model.
