The first thing I will do is put all of the frames into a NumPy array, as NumPy allows for multidimensional arrays. I will do this by:

  1. Traversing the "URL" column of the plaza index
  2. Using the imageio.imread() function to acquire the frames (images) from the URL
  3. Putting those images in a list
  4. Converting that list into a numpy array
In [4]:
all_frames = []
for image in plaza_index["URL"]: # iterating thru urls
    all_frames.append(imageio.imread(image)) # read the image of the url, then append it to array
arr = np.array(all_frames)

Now I will fix the column dimension (i.e. make stationary) but modify all rows, frames, and color channels. I will do this by using a colon (:) to index the multidimensional numpy array. A colon (:) means "select all indices along this axis."

When displaying this new image, the image will be rotated. To fix this, I will rotate it using the transpose() function. If we think of the rotated image's values with axes (0, 1, 2), 0 representing rows, 1 representing columns, and 2 representing color channels, we need to flip the row and column axes to rotate the image.

I will display this newly rotated, up-right image using the matplot imshow() function.

In [5]:
fix_cols = arr[:,:,50,:]
rotate = fix_cols.transpose((1, 0, 2))
plt.imshow(rotate)

To achieve a diagonal slice, I will slice into my NumPy array similar to the way I did it earlier. However, the row and column dimensions of the images are different, so I cannot fix frames (recall formatting: [frames, rows, cols, channels]). Rather, I will fix rows and channels, and manipulate frames and columns.

The final image will come out rotated again, so I will transpose like I did earlier.

In [20]:
cols = []
for i in range(838):
    cols.append(arr[i, :, i, :])
diag = np.array(cols)

rotate2 = diag.transpose((1, 0, 2))
plt.imshow(rotate2)
